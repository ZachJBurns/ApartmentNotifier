### Note:
```bash
This will NOT work on other apartment websites by changing the request URL to another site. You will have to use beautiful soup to get the information from the page you need before running it with a different request URL.
``` 

### Install: 
```bash
git clone "https://github.com/ZachJBurns/ApartmentNotifier.git"
cd ApartmentNotifier
pip install -r requirements.txt
```

### Usage:
```bash
Fill in your Gmail username, password, and recipient emails inside ApartmentNotifier.py

To run the script type in 'python ApartmentNotifier.py'
```
